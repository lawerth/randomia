# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-23

### Added

- Initial release of Randomia library
- Random number generation (`randomRange`, `randomFloat`)
- Basic random selection (`pick`, `pickMultiple`, `sampleWithoutReplacement`)
- Array shuffling with Fisher-Yates algorithm
- Weighted random selection (`weightedPick`, `weightedPickMultiple`, `weightedSampleWithoutReplacement`)
- Probability-based operations (`probability`)
- Object operations (`pickObjectKey`, `pickObjectValue`)
- Hybrid API (class-based and functional)
- Full TypeScript support with type definitions
- Comprehensive test suite (41 tests)
- Detailed documentation and examples
- Error handling with descriptive messages
